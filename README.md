# Breast-Cancer-Classification-using-Deep-Learning-TensorFlow-Keras-
Breast cancer classification dataset and deep learning model built using TensorFlow and Keras. Suitable for medical prediction and machine learning practice.
This repository contains a breast cancer dataset and a deep learning classification model built using TensorFlow and Keras.

The goal of this project is to build a neural network that can classify tumors as malignant (cancerous) or benign (non-cancerous) based on numerical medical features.
